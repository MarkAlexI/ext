Block Site Extension

Block Site Extension - це розширення для браузера, яке дозволяє блокувати небажані сайти, допомагаючи вам зосередитися на важливих завданнях. Розширення призначене для ефективного управління вашим онлайн-часом і збереження продуктивності.

Інструкції по Встановленню

Щоб встановити Block Site Extension, виконайте ці кроки:

  1. Перейдіть до Chrome Web Store або Firefox Add-ons.
  2. Натисніть кнопку "Додати до браузера".
  3. Дотримуйтесь інструкцій для завершення установки.
  4. Перезапустіть браузер, якщо це необхідно.

Часті Питання

Як налаштувати список заблокованих сайтів?

Щоб заблокувати ресурс, введіть частину його імені в лівому полі введення без вказівки протоколу. Наприклад, введення «youtube» заблокує всі субдомени цього ресурсу. У правому полі введення введіть URL-адресу для переспрямування, повністю визначену протоколом, наприклад, "https://google.com". Потім натисніть кнопку зберегти. Якщо URL-адресу перенаправлення залишити порожньою, цільовий сайт буде просто заблоковано.

Чи можу я розблокувати сайти, якщо змінив свою думку?

Так, ви можете розблокувати сайти через меню розширення. Щоб змінити правило, видаліть старе правило та додайте нове.

Чи є можливість імпортувати/експортувати список заблокованих сайтів?

На даний момент розширення не підтримує імпорт та експорт списків заблокованих сайтів. Ми працюємо над цією функцією для майбутніх версій.

Чи є можливість працювати з застосунком не на англійській мові?

Так. На даний момент Block Site Extension підтримує цілий ряд мовних локалей: англійську, українську, французську, німецьку, італійську, арабську, іспанську, корейську, польську, португальську (Бразилія), португальську (Португалія), чеську, японську, китайську спрощену, китайську тайванську, угорську, румунську, шведську, в'єтнамську та гінді (останню - лише для Google Chrome). Немає необхідності шукати "правильний" мовний пакет - він встановиться автоматично під налаштування вашого браузера!
