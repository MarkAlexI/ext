Get Joke Extension

Get Joke Extension - це розширення для браузера, яке надає жарти прямо на вашій панелі інструментів. Лише один клік по іконці встановленного веб-розширення, щоб автоматично в попапі отримати жарт та ще один клік для наступних.

Інструкції по Встановленню

Щоб встановити Get Joke Extension, виконайте ці кроки:

  1. Перейдіть до Chrome Web Store або Firefox Add-ons.
  2. Натисніть кнопку "Додати до браузера".
  3. Дотримуйтесь інструкцій для завершення установки.
  4. Перезапустіть браузер, якщо це необхідно.

Часті Питання

Що робити, якщо я не бачу жартів після установки?

Переконайтесь, що розширення увімкнене в налаштуваннях браузера. Якщо проблема не зникає, спробуйте перезавантажити браузер або перевстановити розширення.

Чи можу я налаштувати тематику жартів?

Поки що розширення не підтримує налаштування тематики жартів. Ми плануємо додати цю можливість у майбутніх версіях.

Чи є версія розширення для інших браузерів?

На даний момент розширення доступне тільки для Chrome та Firefox. Ми розглядаємо можливість розширення підтримки на інші браузери в майбутньому.
